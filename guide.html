INPUTS-Default | Guidance Formatting

For Defaults | Guidance ,the boxed formatting around the value defines if the value is dynamic or based on previous inputs
Non Boxed Value Hardcoded Default | Guidance
Boxed Value Dynamic Default | Guidance.Value is based on previous inputs

Below is the type of Defaults | Guidance and descriptions.
Required --- This is a required input.Usershould enter a valid input
Optional --- This is an optional input.User can leave blank.Mostly Used for ability to estimate multiple items.
Not Used --- This input is not used.Any user input/override will Not be used.
Any other value is a default value calculated in the tool and provided to the user.This default can be overridden.
